# MPK Mini 3 MIDI Script Configurator

An open-source web app for visually configuring an Akai MPK Mini 3 and generating a FL Studio-compatible Python MIDI script.

## What it does
- Assign FL Studio functions to each pad (Bank A & B) and knob
- Generate a ready-to-use device_*.py MIDI script
- Download and drop into your FL Studio hardware folder
